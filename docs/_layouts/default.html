Portfolio Page

This page was developed based on the Freelance Bootstrap theme created by Start Bootstrap and hosted by GitHub Pages. The source code was modified to achieve the following customization:

  • Added the Certification section.
  • Modified the Portfolio section to link to external GitHub Project Pages.
  • Added the audio file to the About section.

GitHub Project Pages

All project pages were developed using the Cayman Jekyll theme. Similar to the Portfolio Page, these web pages are hosted by GitHub Project Pages. The source code was modified to achieve the following customization:

  • Modified the default.html file to a different layout with the "aside" section added to "main".
  • Pagination available in Bootstrap was used to spit content across multiple pages. The JavaScript and jQuery source code available at JSFiddle was used with some modifications to make Pagination work with the new page layout.
